# 飞牛NAS 视频自动转码工具

自动监测文件夹中的视频文件，智能转码为 H.264/H.265 以节省存储空间。专为飞牛OS (fnOS) 设计，支持 Intel Quick Sync (QSV) GPU 硬件加速。

---

## 安装

### 方法一：下载安装包（推荐）

从 [Release](https://github.com/yang1245789/fpk-converter/releases) 下载最新版 `fpkconverter.fpk`，在飞牛 NAS 应用中心手动安装。

### 方法二：自行打包

```bash
git clone https://github.com/yang1245789/fpk-converter.git
cd fpk-converter
fnpack build
# 生成 fpkconverter.fpk
```

---

## 使用

安装后在飞牛OS应用中心启动，点击图标打开 Web UI，或访问 `http://你的NAS_IP:5000`。

---

## 功能特性

- **自动监测** — 定时扫描监控文件夹，发现新视频自动入队转码
- **智能转码** — 自动跳过已处理文件，避免重复转码
- **编码器选择** — 支持 H.264 (libx264) 和 H.265 (libx265)
- **GPU 加速** — Intel Quick Sync (QSV) 硬件加速，大幅降低 CPU 占用
- **分辨率缩放** — 超过 1080p 的视频自动缩放至 1080p
- **智能跳过** — 1080p HEVC 低码率视频自动跳过，避免无效转码
- **数据安全** — 同格式替换时先备份再删除，转码失败自动恢复
- **Web UI** — 可视化配置、实时状态、转码日志、空间统计

## 配置说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 监控文件夹 | 要自动监测的视频目录 | 空 |
| CRF | 质量系数，越小质量越高，范围 1-51 | 23 |
| 编码预设 | 转码速度，越慢压缩率越高 | medium |
| 线程数 | FFmpeg 使用的线程数 | 1 |
| 编码器 | H.264 (兼容性好) / H.265 (更省空间) | H.264 |
| 格式 | 输出容器格式 MP4 / MKV | MP4 |
| GPU加速 | Intel QSV 硬件加速 | 开启 |

## 技术细节

- **扫描模式** — 定时轮询（60秒间隔），不使用 inotify，避免 NAS 内核崩溃
- **串行处理** — 一次只转码一个文件，避免系统过载
- **写入延迟** — 文件加入队列后等待 15 分钟才开始转码，确保文件写入完成
- **临时文件** — 转码临时文件存放在数据分区，支持 30GB 上限自动清理
- **数据库** — SQLite 记录已处理文件，支持 UNIQUE 去重

## 注意事项

1. 首次使用需在 Web UI 中选择监控文件夹并保存配置
2. 转码过程会占用较多 CPU/GPU 资源，建议在空闲时段运行
3. 转码后的文件会替换原文件，请确保重要数据已备份
4. 如需停止转码，点击"停止"按钮即可优雅终止
